# Codex Project Instructions

- Treat this directory as the root for all `yushu_robot` tasks.
- Keep robot part files, URDF files, and meshes in `yushu_robot_urdf/`.
- Keep simulation, training, evaluation, conversion, and controller code in `model/`.
- Update `README.md` when adding a workflow, command, asset, environment, or directory rule.
- When uploading this repository to GitHub, commit and push to `codex` on `https://github.com/Destiny916/yushu_robot.git` unless the user explicitly requests another branch.
